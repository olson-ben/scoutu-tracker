⚙ Google Sheets Sync Setup

One-time setup — takes about 5 minutes.
You only do this once. After setup, every save auto-syncs to your Google Sheet, visible on any device.
Step 1 — Create your Google Sheet
Go to sheets.google.com → click Blank → rename it "ScoutU Tracker".
Copy the Sheet ID from the URL:
docs.google.com/spreadsheets/d/THIS_PART/edit
Step 2 — Get a Google API Key
1. Go to console.cloud.google.com
2. Create a new project (name it anything)
3. Go to APIs & Services → Library → enable Google Sheets API
4. Go to APIs & Services → Credentials → Create Credentials → API Key
5. Copy the key and paste below
Step 3 — Make your Sheet writable
In your Google Sheet go to Share → under "General access" set to "Anyone with the link" → Editor.
This lets your API key write data to it.
